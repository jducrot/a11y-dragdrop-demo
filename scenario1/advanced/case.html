Advanced Scenario: Word Bank With Dropdown

There are three ways one can interact with the elements in this exercise:

  1. Mouse users can drag states from the word bank and drop them onto the correct sentences.
  2. Keyboard users can use a menu next to each dropzone to select which state to place in that sentence.

Note: The menu only becomes visible when it receives keyboard focus. It you would rather the menu buttons remain visible at all time, please toggle the "Show Menu Buttons" checkbox below.
