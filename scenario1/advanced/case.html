Advanced Scenario: Word Bank With Dropdown

Instructions

There are three ways one can interact with the elements in this exercise:

  1. Mouse users can drag the words from the word bank and drop them into the correct missing word spots inside the sentences in the exercise area.
  2. Keyboard users can select each word in the word bank using Enter or Spacebar, then focus the desired target location using Tab or Shift + Tab, and finally drop the selected state on the target by pressing Enter or Spacebar.
  3. Both mouse and keyboard users can open a menu next to each dropzone to select which state to place in that dropzone.
